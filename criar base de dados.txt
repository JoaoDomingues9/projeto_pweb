criar base de dados:

createdb -U postgres receitas

depois:
psql -U postgres receitas

CREATE TABLE registo(id_registo SERIAL PRIMARY KEY, nome VARCHAR(100), sobrenome VARCHAR(100), email VARCHAR(200), password VARCHAR(100));

CREATE TABLE receitas (
  id_receitas SERIAL PRIMARY KEY,
  id_registo INTEGER REFERENCES registo(id_registo)
);

CREATE TABLE ingredientes (
  id_ingredientes SERIAL PRIMARY KEY,
  id_receitas INTEGER REFERENCES receitas(id_receitas),
  nome_receita VARCHAR(50),
  ingredientes VARCHAR(4000)
);

Inserir dados:
INSERT INTO registo(nome, sobrenome, email, password) VALUES('joao', 'domingues', 'joao@gmail.com', 'ola');

depois abre o pgAdmin4, mete a password que definites antes e pronto.
